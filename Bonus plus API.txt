1. Для получения списка транзакций указывается 
	POST /retail/transactions 
с фильтром по дате
2. Для получения информации о конкретной продаже для каждой полученной продажи нужно выполнить запрос
	GET /retail/{id}
3. Для получения информации о клиенте нужно выполнить запрос с номером телефона клиента, взятого из ответа запроса GET /retail/{id}
	GET /customer?phone={phone}

Каждый запрос сопровождается заголовком
Authorization: ApiKey D413AE47-D597-4D4F-9987-3172C2632402 (в данном случае ключ такой)
